INTRODUCTION

The files in this folder provide examples for benchmarking SigLib functions
using the XMOS toolset.

The XMOS timer functions provide cycle accurate measurements for the code
execution time.

Please read the appropriate section of the SigLib reference manual for
further information.

Android Studio can be downloaded from https://developer.android.com/studio/index.html.
