SigLib File I/O Example Programs

  

This folder contains the SigLib file I/O example programs.

Source Files

    bin2wav.c - Convert SigLib binary file to mono .wav format
    dat2wav.c - Convert SigLib .dat file to mono .wav format
    sig2csv.c - Convert SigLib .sig file to comma separated variables file (.csv)
    bin2wav.c - Convert SigLib .bin file to mono .wav format
    raw2wav.c - Convert raw PCM file to mono .wav format
    sig2wav.c - Convert SigLib .sig file to mono .wav format
    sig2wav_scaled.c - Convert SigLib .sig file to mono .wav format. Scale data to 0 dB FS
    sigplot.c - Graphically plot SigLib .sig file
    test_bin.c - Binary file I/O example
    test_csv.c - Comma separated variables file I/O example
    test_dat.c - SigLib .dat file I/O example
    test_raw.c - Raw (PCM) file I/O example
    test_sig.c - SigLib.sig file I/O example
    test_wav.c - Mono .wav file I/O example
    wav2dat.c - Convert mono .wav file to SigLib .dat
    wav2sig.c - Convert mono .wav file to SigLib .sig
    wavinfo.c - Display .wav file header information
    wavplot.c - Plot graph of mono .wav file data
    wavplot_stereo.c - Plot graph of stereo .wav file data
    xmt2wav.c - Convert .xmt file to mono .wav format

The batch files supplied work with the GCC and Microsoft Visual Studio compiler (64 bit versions supplied) however the example programs can be built using almost any 32 or 64 bit compiler.
