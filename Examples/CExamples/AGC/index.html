SigLib Automatic Gain Control Example Program

  

The files in this folder provide examples of how to use the Automatic Gain Control functions.
This program uses the envelope tracking AGC algorithm which is recommended for applications such as speech.

Rudyard Kipling If source Librivox Version 10, Martin Clifton: https://librivox.org/if-by-rudyard-kipling/

The batch files supplied work with the GCC and Microsoft Visual Studio compiler (32 and 64 bit versions supplied) however the example programs can be built using almost any 32 or 64 bit compiler.

Filename    Description
wav_process_envelope.c    Automatic Gain Control.



Executing The Example

 The following command processes the supplied .wav file under Linux:
./process_wav_agc_envelope.sh Kipling_If_48kHz_Attenuated.wav

process_wav_agc_envelope.sh plays out the processed audio file using aplay.

This folder also includes a python script, plot_debug_log.py that can plot the internal parameters of the algorithm, provided that SIGLIB_ENABLE_DEBUG_LOGGING is set to '1' when SigLib is rebuilt.
This file is executed as follows:
python plot_debug_log.py siglib_debug.log
This file requires the following Python libraries to be installed: numpy, pandas, matplotlib and sys.
