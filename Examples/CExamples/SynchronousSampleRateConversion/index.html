SigLib Synchronous Sample Rate Conversion Example Programs

  

The files in this folder provide examples of how to use the synchronous sample rate conversion functions.

The input .wav file is converted from 16 kHz to 48 kHz then this file is then down converted back to 16 kHz.

Shell scripts are supplied for GCC (./gbr.sh) and batch files for Microsoft Visual C/C++ (mbr.bat).
